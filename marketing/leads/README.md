# Tournable — база клиентов и аутрич

Материалы для сбора клиентской базы и работы с ней (подзадачи APP-10 … APP-16).

## Файлы
- **[leads-database.csv](leads-database.csv)** — база лидов по 7 категориям, готова к импорту в CRM (HubSpot / Notion / Google Sheets). Колонки: Категория, Подзадача, Название, Город, Профиль, Контактное лицо, Телефон, Email, Сайт, Instagram, Telegram, Приоритет, Статус, Источник, Заметки.
- **[outreach-playbook.md](outreach-playbook.md)** — как работать с базой: настройка CRM, приоритеты, каналы контакта, шаблоны переписки, скрипт холодного звонка, воронка, метод расширения базы.

## Быстрый старт
1. Импортируй `leads-database.csv` в HubSpot Free (или Notion) — инструкция в плейбуке, раздел 1.
2. Отфильтруй «Приоритет = Высокий» — начни с любительских лиг Астаны (LLF, ASTFL, LFA) и SPORT PROJECTS.
3. Первое касание — по каналу из плейбука (раздел 3): лигам — Instagram/WhatsApp, федерациям — email+звонок, спортшколам — звонок.
4. Каждый контакт двигай по воронке (раздел 7), отмечай статус в CRM.
